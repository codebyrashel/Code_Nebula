# Code Nebula

Code Nebula is an online judge platform inspired by LeetCode, CodeChef, and Codeforces. The project focuses on building a clean, modern frontend for evaluating programming solutions.

---

## How to Run Locally

### Frontend

`Prerequisites`

- Node.js (v18 or later)
- npm

#### Steps
```
git clone https://github.com/your-org/Code_Nebula.git

  cd code_nebula

  npm install

  npm run dev

  http://localhost:5173
```
## Tech Stack
- React (Vite)
- Tailwind CSS
- Tailwind Prettier Plugin

> Note: Code Nebula is currently in its early development phase. The tech stack may evolve over time, with additional tools or technologies introduced as the project grows and requirements become more defined.
